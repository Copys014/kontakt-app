1. Otvori GitHub i kreiraj novi repozitorijum, npr. `kontakt-app` (public).
2. Klikni 'Add file' → 'Upload files'.
3. Ubaci fajlove iz ovog ZIP-a: index.html, style.css, main.js.
4. Klikni 'Commit changes'.
5. Idi na Settings → Pages.
6. Izaberi Branch: `main`, Folder: `/ (root)` i klikni Save.
7. Poseti: https://<tvoj-username>.github.io/kontakt-app/
